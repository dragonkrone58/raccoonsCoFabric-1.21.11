Creating Basic Item checklist:

1. register item in ModItems & add to item group(s)
2. Datagen ModModelProvider & generate json file(s)
3.  a. generate recipe
    b.
4. Datagen generate & build lang name


Creating Basic Block checklist:

1. register block & block item in ModBlocks & add to item group(s)
2. Datagen & generate:
    a. ModBlockTagProvider if applicable
    b. ModModelProvider for block state models
    c. ModRecipeProvider & ModLootTableProvider
    d. ModLanguageProvider


Creating Basic Flower checklist:
1. register block, block item & flower pot block in ModBlocks & add to item group(s)
2. Datagen & generate flower & flower pot:
    a. ModBlockTagProvider if applicable
    b. ModModelProvider for block state models
    c. ModRecipeProvider & ModLootTableProvider
    d. ModLanguageProvider
3. RaccoonsCoClient: add block renderer map layer
4. Generate in World:
    a. ModConfiguredFeatures: Register key and in bootstrap
    b. ModPlacedFeatures: Register key and in bootstrap
    c. ModFlowerGeneration: define parameters to generation


Creating Basic Armor Checklist:
1. register armor in ModItems
2. define key, durability & armor material in ModArmorMaterials
3. Datagen & generate:
    a. ModModelProvider
    b. ModRecipeProvider
    c. ModItemTagProvider
    d. ModLanguageProvider
4. Manually add JSON file to resources/assets/raccoonsco/equipment
5. add textures to resources/assets/raccoonsco/textures/entity/equipment/humanoid & /humanoid_leggings
6. add item textures

ctrl + H to see class hierarchies & browse vanilla classes to see functionalities.
